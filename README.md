# Nextjs Boilerplate

## Features

- Nextjs
- Typescript
- Bundle Analyzer

- styled-components

- next-sitemap
- next-seo

- Eslint
- Prettier
- Stylelint

- Husky
- Lint staged

- Vercel

## Customization

1. customize name in `package.json`
2. customize siteUrl in `next-sitemap.config.js`
3. customize favicon in `src/app/favicon.ico`
